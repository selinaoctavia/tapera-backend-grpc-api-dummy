# Gore API Boilerplate

## this boilerplate using

- [Gorilla Mux](https://www.gorillatoolkit.org/pkg/mux "HTTP Toolkit")
- [GORM](https://gorm.io/ "ORM")
- [SWAGO](https://github.com/swaggo/swag "Swagger")
- [SqlMocking](https://github.com/DATA-DOG/go-sqlmock/tree/v1.4.1 "SqlMocking")
- [SuiteTesting](https://github.com/stretchr/testify "SuiteTesting")
- [Zerolog](https://github.com/rs/zerolog "Logger")
- [Lumberjak](https://github.com/natefinch/lumberjack "Logger file writer and rotator")
- [Validator](https://godoc.org/gopkg.in/go-playground/validator.v9 "Struct field validator")
